Initial GitHub Push

1. Create rebository folder locally
2. Open a new repository in GitHub
3. Link offline Repository (webpage folder) to GitHub repository
    VSCode terminal (from repository folder):
        git init
        git remote add origin https://github.com/SpikeMurphy/REPOSITORYNAME.git
        git remote -v
            origin  https://github.com/SpikeMurphy/REPOSITORYNAME.git (fetch)
            origin  https://github.com/SpikeMurphy/ConformationLabWeb.git (push)
        git add .
        git commit -m "Initial commit: ConformationLabWeb"
        git branch -M main
        git push -u origin main
